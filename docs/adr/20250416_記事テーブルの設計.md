# Decision

以下の設計にする
```mermaid
erDiagram
articles {
  bigint id pk
 varchar(10) media "qiita | zenn"
  varchar(100) title
  varchar(30) author
  varchar(255) description
  text url
  datetime created_at
}
```


## Reason

- urlは数年に一度長さの上限値が増えているのでtext型にする
- updated_atはupdateを行うことがないので不要
- description, author, titleは感覚値で設定
